Claude Code not detected

      

        
Yunomia needs the Claude Code CLI. It runs each agent as a real Claude Code session, so it can't operate without it.

        
Install Claude Code (one-time, ~2 min)

        
    
          
  1. Open claude.com/claude-code and follow the install instructions for your OS.
            
    Quick path on macOS / Linux (paste in Terminal):
              curl -fsSL https://claude.ai/install.sh | bash
            
    
          
    2. 
          
  2. Sign in to Claude Code with your Anthropic account once installation finishes (the installer will prompt you).
    3. 
          
  3. Quit Yunomia (⌘Q) and reopen from Applications.
    4. 
        

        
Yunomia checks for Claude Code each time it launches. Once installed, this dialog stops appearing.

        
If Claude Code IS already installed

        
    
          
  1. Open Terminal and run: which claude
    2. 
          
  2. Click Re-check below. If still not detected, the binary is in a non-standard location - we'll add a manual-path option in a future release.
    3.
